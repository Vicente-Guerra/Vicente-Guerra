## 🚀 Proyectos Destacados

### 🤖 Sistema de Control de Brazos Robóticos (IoT / Robótica)
*Ecosistema de hardware y software para automatizar movimientos físicos.*
*   **Stack:** `C++`, `ESP32`, `WebSockets`, `HTML/CSS/JS`.
*   **Logro:** Desarrollé un servidor web embebido en un ESP32 que permite grabar secuencias de movimientos en tiempo real, almacenarlas en memoria y hacer que los brazos las repliquen de forma autónoma.

### 📊 Conversor Corporativo CSV a Excel (Automatización / GUI)
*Aplicación de escritorio para la optimización de flujos de trabajo administrativos.*
*   **Stack:** `Python`, `CustomTkinter`, `Pandas`.
*   **Logro:** Diseñé un programa con interfaz gráfica moderna que automatiza la conversión de reportes CSV en archivos Excel, aplicando estilos y reglas de negocio específicas de la empresa.

### 🛡️ Scripts de Hacking Ético y Seguridad (Ciberseguridad)
*Herramientas para auditorías de seguridad y automatización de pruebas.*
*   **Stack:** `Python`, `Redes (TCP/IP)`.
*   **Enfoque:** Monitoreo de nuevos exploits y desarrollo de scripts propios para entender y mitigar fallos de vulnerabilidad en sistemas y redes.
